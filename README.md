This project serves as a template for api backend using flask. It has two modes, which are local dev mode and remove deploy mode runing inside Dokcer contianer.

## How to run the project

# On your loca env

First enable your virtual environment

```sh
python3 -m venv .venv
```

Activate the virtual environment

```sh
. .venv/bin/activate
```

Run the project locally

```sh
flask --app main run --debug
```
